# Hospice Dev

We keep things alive a little longer than they'd otherwise survive.

---

## What we do

This organisation maintains **[Konine](https://github.com/hospicedev/konine)** — a PHP 8.4+ HMVC framework and the final chapter in the Kohana lineage.

**Kohana → Koseven → Konine.**

Kohana was an elegant, opinionated PHP framework with a devoted following. When it went end-of-life, the Koseven project kept it running through PHP 7 and into PHP 8. When Koseven was archived in April 2026, Konine took over — not to rebuild it, not to extend it, but simply to keep existing applications running while their owners plan a proper migration.

Konine is **maintenance and security fixes only**. No new features. No new direction. Just compatibility patches and security updates for as long as PHP 8.x remains in active support — or until the maintainers are no longer able to continue, whichever comes first.

---

## Who this is for

Developers with **existing Kohana or Koseven applications** who need more time — time to plan a migration, time to raise budget, time to find the right moment. Konine buys that time without forcing you off a working stack before you're ready.

If you're starting a new project, please look at [CodeIgniter](https://codeigniter.com/) instead. It shares the same spirit as Kohana and is actively maintained.
